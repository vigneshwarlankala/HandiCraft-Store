# HandiCraft-Store

HandiCraft-Store is a web-based e-commerce platform designed to showcase and sell handmade craft products online. The project aims to support local artisans by providing a digital marketplace where users can explore unique handcrafted items through a simple and responsive interface.

---

## 🚀 Features

- 🛍️ Product showcase for handmade craft items  
- 📂 Category-based product browsing  
- 📱 Fully responsive design for all devices  
- 🎨 User-friendly and modern interface  
- 🔍 Easy navigation and product viewing  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript

## ⚡ Getting Started

1. Clone the repository

```bash
git clone https://github.com/vigneshwarlankala/HandiCraft-Store.git
```

2. Navigate to the project folder

```bash
cd HandiCraft-Store
```

3. Open `index.html` in your browser
